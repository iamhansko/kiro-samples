# 제품 개요

Country Quiz는 플레이어가 3D 지구본에서 국가를 식별하는 인터랙티브 지리 퀴즈 게임입니다. 플레이어에게 국기가 표시되면 지구본을 회전하고 클릭하여 해당 국가를 찾아야 합니다. 게임은 점수를 추적하고 필요할 때 힌트를 제공합니다.

## 핵심 기능

- 국경선이 표시된 인터랙티브 3D 지구본
- 국기 기반 국가 식별
- 실시간 좌표 표시 (위도/경도)
- 점수 추적 시스템
- 국가 위치를 보여주는 힌트 시스템
- 나침반 방향 표시 (동/서/남/북)
- 드래그 방향 설정 가능
- 위도/경도 선이 표시된 시각적 십자선

## 사용자 상호작용

플레이어는 드래그로 지구본을 회전하고, 마우스 휠로 줌을 조절하며, 십자선을 목표 국가 위에 위치시킨 후 답을 제출합니다. 게임은 즉각적인 피드백을 제공하고 정답일 경우 자동으로 다음 국가로 진행합니다.
