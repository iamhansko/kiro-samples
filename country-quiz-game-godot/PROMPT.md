무작위로 표시된 국기를 보고 3D 지구본을 마우스 드래그로 회전시켜 국기에 해당하는 국가를 맞추는 Godot 엔진 기반의 게임을 제작합니다.

# 요구사항
- 게임 시작 시 국기 무작위로 표시
- 사용자는 마우스로 지구본을 회전하며 국가 탐색
- 사용자는 십자선을 국가에 맞추고 Submit 버튼 클릭
- 정답이면 Score 증가 및 다른 국기 무작위로 표시
- Hint 버튼 클릭하면 국기에 해당하는 국가 위치로 이동
- Reset 버튼 클릭하면 게임 초기화

# 기술 스택
- Godot Engine 4.5 / GDScript
- Forward Plus Renderer
- SphereMesh
- ImmediateMesh
- StandardMaterial3D
- Nearest
- GeoJSON

# 제약사항
- C#, C++ 사용 금지
- 네이티브 Godot 기능을 제외한 외부 플러그인 사용 금지

# 예상 결과물
- 프로젝트 이름: "Country Quiz Game"
- 해상도: 1280x720
- 메인 씬 경로: `res://scenes/main.tscn`
- 반지름 길이 2의 흑색 구체 mesh 생성하고 64개 radial segments, 32개 rings로 부드러운 3D 지구본 표현
- 마우스 좌클릭 드래그로 3D 지구본 위도/경도 기반 회전
- 마우스 휠로 3D 지구본 줌 인/아웃 (거리: 2.1 ~ 10.0)
- 줌 인/아웃 거리에 비례하여 마우스 드래그 회전 감도 조정 
- 마우스 좌클릭 드래그 방향 반전 옵션 (Direction 토글 버튼, 기본값 : 활성화)
- `public/earth.geo.json` 파일로부터 GeoJSON 형식의 국가별 좌표 읽고 백색으로 국경선 파싱 (LINE_STRIP 프리미티브, 지구본 표면 +0.005 위에 렌더링하여 겹침 방지)
- `public/flags/` 폴더로부터 국기 이미지 불러오기 (국기 이미지 파일명 : ISO 2자리 코드 iso_a2 기준, 예시: kr.svg, us.svg)
- 게임 화면 중앙에 녹색 십자선 표시, 십자선이 가리키는 위치의 위도/경도 실시간으로 계산하여 표시
- 십자선이 가리키는 위치에 해당하는 위도/경도의 국가명 표시
- 북극점/남극점에 반지름 길이 0.02 백색 구체 마커 표시
- 적도선/본초자오선을 점선 형태로 표시 (dash: 2, gap: 2)
- 게임 화면 가장자리에 N, S, E, W 나침반 레이블 동적 배치, 지구본 회전에 따라 실시간으로 나침반 레이블 위치 조정
- `public/flags/` 폴더로부터 국기를 무작위로 선택하고 게임 화면에 선택된 국기를 표시
- 사용자는 십자선으로 국가를 가리키고 Submit 버튼 클릭하여 십자선이 가리킨 국가와 국기에 해당하는 국가의 ISO 2자리 코드 iso_a2가 일치하는지 확인:
- Submit 버튼: (정답) 일치할 경우 Score +1 증가, "Correct" 표시 (녹색), 1초 후 다음 국기 무작위로 표시
- Submit 버튼: (오답) 불일치할 경우 "Wrong" 표시 (빨간색), 1초 후 원래 상태로 복귀
- Score 레이블은 게임 화면 좌측 상단에 표시
- Reset 버튼: 게임 초기화 (점수 0, 새 국기 무작위로 표시)
- Hint 버튼: 정답 국가의 중심 좌표로 지구본 이동
- 위도/경도를 3D 구면 좌표로 변환: x = radius × cos(lat) × sin(lon)
- 위도/경도를 3D 구면 좌표로 변환: y = radius × sin(lat)
- 위도/경도를 3D 구면 좌표로 변환: z = radius × cos(lat) × cos(lon)
- Point-in-Polygon 알고리즘으로 좌표가 속한 국가 판별
