웹 기반 2인용 알까기 게임을 제작합니다. HTML, CSS, JavaScript를 사용하며, 물리 엔진과 충돌 감지를 포함한 완전한 게임 로직을 구현합니다.

# 요구사항
- 플레이어는 자신의 말을 마우스로 드래그하여 방향과 힘 조절
- 마우스로 보드(캔버스) 경계를 벗어나 보드(캔버스) 밖에서 드래그하더라도 드래그 방향과 힘 조절 가능
- 마우스를 놓으면 말이 반대 방향으로 튕겨나감
- 말끼리 충돌 시 물리 법칙에 따라 운동량 전달
- 보드 경계에 닿은 말은 제거됨
- 모든 말이 정지하면 플레이어 공격권 자동 전환
- 상대방 말을 모두 제거하면 라운드 승리 
- 4라운드 중 더 많은 라운드를 승리한 플레이어가 최종 승리
- 4 라운드 후 최종 승자 결정 및 게임 자동 리셋
- 게임 리셋 버튼 구현

# 기술 스택
- HTML5: 시맨틱 마크업, 한국어 지원(lang="ko")
- CSS3: 커스텀 속성(변수), Flexbox 레이아웃
- Vanilla JavaScript: 프레임워크 없음, ES6+ 문법 사용

# 제약사항
- 외부 라이브러리 사용 금지 (jQuery, React 등)
- 빌드 도구 사용 금지 (Webpack, Vite 등)

# 예상 결과물
- 파일 : index.html, style.css, script.js
- 보드 크기: 480px x 480px 캔버스, 18 x 18 그리드
- 플레이어: 2명 (P1: 청색, P2: 적색)
- 말 개수: 플레이어 당 51개 (칸 중앙이 아닌 점에 3 × 17 배치)
- 라운드: 4라운드 3선승제, 2승 2패일 경우 무승부 처리
- 라운드: 4라운드 중 1, 3 라운드는 플레이어 1 선공
- 라운드: 4라운드 중 2, 4 라운드는 플레이어 2 선공 
- 말 충돌 조건: 말 간 거리 20px 미만 시 충돌
- 말 반지름: 10px
- 말 중앙에 번호 표시 (1-51)
- 플레이어 1 말: 파란색 원 (#2196F3), 테두리 (#1976D2)
- 플레이어 2 말: 빨간색 원 (#F44336), 테두리 (#C62828)
- 플레이어가 선택한 말 주위에 금색 원 (#FFD700), 반지름 15px, 선 두께 4px
- 플레이어가 말 드래그 시 금색 화살표(#FFD700) 표시, 말이 튕겨나갈 방향 (드래그 반대)
- 말 파워 미터: 드래그 거리에 비례 (최대 100%)
- 실시간 정보 표시: Round(현재 라운드 / 총 라운드)
- 실시간 정보 표시: Score(각 플레이어 라운드 승수)
- 실시간 정보 표시: Remaining(각 플레이어 남은 말 수)
- 실시간 정보 표시: Current(현재 공격권을 가진 플레이어)
- 실시간 정보 표시: Power(플레이어가 선택한 말의 파워 미터)
